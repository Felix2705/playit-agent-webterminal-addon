# Playit Agent WebTerminal Add-on (0.0.1-Beta)

Dieses Home-Assistant Supervisor Add-on startet automatisch den **Playit Agent** und stellt zusätzlich eine **Web-Terminal-UI (ttyd)** bereit.

## Web-Terminal nutzen
1. Add-on installieren und starten
2. In Home Assistant das Add-on im **Ingress/Sidebar** öffnen
3. Dort die Shell-Befehle eingeben

## Add-on Dateien
- `playit_agent_webterminal/` enthält `config.yaml`, `Dockerfile`, `run.sh`, `start_playit.sh`, `build.yaml`, `icon.svg`
